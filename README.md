# NobleX

A Modern Programming Language built on top of the Powerful Java Virtual Machine, 
also known as JVM

![Logo](Noble.png)

## Syntax
### Conditionals:
#### if-else-elif
```NobleX
int age = 13;
if(age >= 7 & age < 13){
 output.println("You are a child");
} elif(age >= 13 & age < 18){
 output.println("You are a teenager");
} elif(age => 18){
 output.println("You are a Adult");
} else{
 output.println("Your Age Group is not Provided");
}
```
